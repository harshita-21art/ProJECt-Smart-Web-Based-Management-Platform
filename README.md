# ProJECt: Smart Web-Based Management Platform

ProJECt is a web-based platform designed to provide an efficient and user-friendly system for managing digital records and interactions. Built using PHP, MySQL, and XAMPP, the platform focuses on simplicity, responsiveness, and structured data management. It provides a centralized system where users can securely access and manage records while administrators oversee system operations through a dedicated dashboard.

## ✨ Features

### For Users
* **Simple Interface:** Clean and responsive user interface for easy navigation.
* **Secure Login System:** Authentication system ensuring controlled and secure access.
* **Data Management:** Ability to store, view, and manage digital records efficiently.

### For Administrators
* **Admin Dashboard:** Centralized dashboard to monitor and manage system activities.
* **User Management:** Add, update, or remove users and manage their records.
* **Database Integration:** Reliable storage and retrieval of system data through MySQL.

### System Features
* **Responsive Design:** Works smoothly across desktops, tablets, and mobile devices.
* **Fast Performance:** Optimized backend and frontend for quick loading.
* **Structured Data Handling:** Organized database schema ensuring data reliability and integrity.

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP (Apache Server via XAMPP)
* **Database:** MySQL
* **Tools:** Git, GitHub, XAMPP
* **Environment:** Localhost Development using XAMPP

## 🚀 Getting Started

Follow these steps to run the project locally.

### Prerequisites
* XAMPP (Apache & MySQL enabled)
* Web Browser (Chrome, Edge, Firefox, etc.)
* Git (optional for cloning the repository)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ashwikarai22-droid/ProJECt.git
   ```

2. **Move the project to the XAMPP directory**

   Copy the project folder to:

   ```
   xampp/htdocs/
   ```

3. **Start XAMPP Services**

   Open the **XAMPP Control Panel** and start:

   * Apache
   * MySQL

4. **Import the Database**

   * Open **phpMyAdmin** in your browser:
     ```
     http://localhost/phpmyadmin
     ```
   * Create a new database (e.g., `project_db`)
   * Import the provided `.sql` file from the `database/` folder.

5. **Run the Project**

   Open your browser and go to:

   ```
   http://localhost/ProJECT
   ```
